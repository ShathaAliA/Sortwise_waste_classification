# Sortwise_waste_classification
AI-Powered Waste Classification System for Household Recycling Support.

----------------------------------------
## Dataset
TrashNet - 2,527 images across 6 classes (Glass, Paper, Cardboard, Plastic, Metal, Trash).

----------------------------------------
## Model
MobileNetV2 with Transfer Learning. Test Accuracy: 85%.
